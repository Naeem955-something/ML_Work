Task Definition

What This Model Does

Classifies social media posts written in Bengali, English, or Banglish (code-switched) into one of three toxicity categories.

0
Explicitly Harmful
1
Subtly Harmful
2
Neutral
📐
Competition Metric: F1NOP = Macro F1 / (Trainable Parameters + ε)
Frozen embeddings keep 9M parameters out of the NOP denominator, leaving only ~10,851 — directly maximising F1NOP.
Model Design

Architecture

[Text Input]
↓
[Frozen FastText Embeddings]300-dim · English + Bengali · NOP = 0
↓
[3 × Conv1d Layers]kernel sizes 2, 3, 4 → bigram / trigram / 4-gram detectors
↓
[Global Max Pooling]position-invariant feature extraction
↓
[Dropout 0.5 → Linear]only trainable layers → ~10,851 params counted in NOP
↓
[Softmax Output]3-class probabilities
ComponentDetail
EmbeddingsMUSE-aligned FastText (EN + BN) · 300-dim · frozen
EncoderTextCNN · kernel sizes [2, 3, 4] · 4 filters each
Trainable Parameters~10,851 (frozen embeddings excluded from NOP)
LossWeighted CrossEntropy + Label Smoothing (0.1)
RegularizationDropout (0.5) · AdamW weight decay (0.05)

Notebook Walkthrough

Step-by-Step Description

Step 1ImportsAll libraries loaded in one place — no NameError mid-run.
Step 2Install thopNOP profiling library required by the official competition scorer.
Step 3ConfigurationAll paths and hyperparameters in one cell — change once, propagates everywhere.
Step 4Random SeedsSeeds fixed across Python, NumPy, and PyTorch (CPU + GPU) before any data loading.
Step 5Load DatasetReads CSVs, prints class distribution — reveals imbalance that informs Step 10.
Step 6Language DetectionUnicode block range detection for Bengali / Banglish / English — confirms need for bilingual embeddings.
Step 7PreprocessingRemoves URLs, @mentions, #symbols. Lowercases ASCII only — Bengali script has no case.
Step 8Build VocabularyTop-30,000 tokens from training corpus with <PAD> and <UNK> special tokens.
Step 9Load EmbeddingsMUSE-aligned 300-dim FastText vectors for English + Bengali into one shared matrix.
Step 10Class WeightsInverse-frequency weights so minority classes are penalised more in the loss.
Step 11Dataset & LoadersDynamic padding per batch — sequences padded to batch max, not global max.
Step 12TextCNN ModelFrozen embeddings + 3× Conv1d (k=2,3,4) + dropout + linear output. NOP ≈ 10,851.
Step 13Optimizer & LossAdamW + Weighted CrossEntropy (label smoothing 0.1) + ReduceLROnPlateau.
Step 14Training LoopUp to 20 epochs, mixed-precision, early stopping (patience=5), best checkpoint auto-saved.
Step 15Visualizations4-panel plot: loss curves, macro F1, per-class F1, confusion matrix.
Step 16Classification ReportPrecision / recall / F1 per class from best validation checkpoint.
Step 17Test PredictionsLoads best weights, runs inference on full test set.
Step 18SubmissionNOP verified with thop.profile(), CSV format validated, submission.csv saved.
